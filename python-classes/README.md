# Python - Classes and Objects

This project covers the basics of Object-Oriented Programming in Python:
- Classes and instances
- Public, protected, and private attributes
- Properties and getter/setter methods
- Class documentation and PEP8 style

## Files

| File | Description |
| --- | --- |
| 0-square.py | Empty Square class |
| 1-square.py | Square with private size attribute |
| 2-square.py | Square with size validation |
| 3-square.py | Square with area() method |
| 4-square.py | Square with size property |
| 5-square.py | Square with my_print() method |
| 6-square.py | Square with position property |
